# voice-agent-animations

Static preview site for voice agent status animations.

## Run locally

From this directory:

```sh
python3 -m http.server 4173 --bind 127.0.0.1
```

Then open:

- Focus view: http://127.0.0.1:4173/
- All animations view: http://127.0.0.1:4173/#all

Stop the server with `Ctrl+C`.
